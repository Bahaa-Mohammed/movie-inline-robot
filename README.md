# Movie Info Bot
A simple movie information finder telegram bot

---

## Variables

- `API_HASH` Your API Hash from my.telegram.org
- `API_ID` Your API ID from my.telegram.org
- `BOT_TOKEN` Your bot token from @BotFather

---

## Commands

```
start - For start bot
info - For movie information
```
